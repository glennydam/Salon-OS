# Assets

Demoen bruger midlertidige eksterne billeder fra Unsplash, så repositoryet forbliver let og kan deployes gratis.

## Udskiftning med Super Cuts egne billeder

Læg godkendte billeder i denne mappe og opdatér billedstierne i `index.html` og `app.js`.

Anbefalede filnavne:

- `hero-supercut.webp` — vertikal eller bred hero, mindst 2000 px
- `service-colour.webp` — balayage/blond
- `service-cut.webp` — klip og styling
- `service-extensions.webp` — extensions
- `before.webp` og `after.webp` — samme vinkel og lys
- `valdeta-working.webp` — ejeren i arbejde
- `customer-maria.webp` — anonymiseret demoportræt eller behandlingsresultat

Brug kun billeder, som salonen og eventuelle afbildede kunder har godkendt til webbrug. Eksportér som WebP, sRGB, kvalitet 78–85.
