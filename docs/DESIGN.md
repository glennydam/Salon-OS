# Design direction

## Kreativ retning

Varm skandinavisk editorial møder præcis software. Kundeoplevelsen skal føles som et premium livsstilsbrand; Salon OS skal føles som et roligt cockpit.

## Principper

1. Ét tydeligt fokus pr. skærm.
2. Typografi og billeder bærer udtrykket.
3. Luft er funktionelt.
4. Bevægelse forklarer — den pynter ikke.
5. Kunden skal føle tillid før teknologi.

Display: Italiana. UI og brødtekst: DM Sans.

## Billeder

- Naturligt lys, ægte tekstur, rolig beskæring.
- Hår og håndværk frem for modelpositur.
- Ingen pink beauty-klichéer, glitter eller åbenlyse AI-ansigter.
- Super Cuts egne billeder skal erstatte placeholders efter skriftlig godkendelse.

## Bevægelse

- 180–750 ms afhængigt af elementets størrelse.
- Ease-out, ingen bounce.
- Respektér `prefers-reduced-motion`.
- Ingen konfetti.

## Tilgængelighed

- Synlig tastaturfokus via browserstandarder og kontrast.
- Mindste trykflade cirka 44 px.
- Semantiske knapper, labels og `aria-live`.
- Indhold forbliver forståeligt uden animation.
